# itsumono

モバイルファーストの買い物リストWebアプリ。サーバー不要の単一HTMLファイルで構成され、LocalStorageでデータを永続化する。

## 機能

- **2タブ構成**
  - **ホームタブ**: 全アイテムをカテゴリ別に表示。追加・編集・削除・ストアへの送信が可能。カテゴリは初期状態で折りたたみ。
  - **ストアタブ**: 買い物中にチェックして購入済みとして記録
- **カテゴリ管理**: 16種類のデフォルトカテゴリ（絵文字付き）、カスタムカテゴリの追加・編集・削除が可能
- **購入日時の記録**: ストアタブでチェックした商品の購入日時を記録、ホームタブで購入日時順にソート表示
- **バックアップ/レストア**: JSONファイルによるデータのエクスポート・インポート（ハンバーガーメニューから実行）
- **ダークモード**: システム設定（prefers-color-scheme）に連動
- **iPhone対応**: PWA表示、ズーム回避（入力欄16px以上）、タッチターゲット44px以上、ノッチ対応（safe-area-inset）

## データ構造（LocalStorage: `itsumono_data`）

```json
{
  "items": [
    { "id": "item_...", "name": "商品名", "categoryId": "cat_...", "inStore": true, "purchaseDate": "2026-04-30T..." }
  ],
  "categories": [
    { "id": "cat_...", "name": "カテゴリ名", "emoji": "🥬" },
    { "id": "__none__", "name": "未分類", "emoji": "" }
  ]
}
```

## 開発・確認方法

ビルド工程なし。index.html をブラウザで直接開く、または簡易HTTPサーバーを起動して確認する。

```bash
python3 -m http.server 8000
# http://localhost:8000 でアクセス
```

## iPhone / PWA 対応詳細

- **ビューポート**: `viewport-fit=cover` を指定し、iPhone X以降のノッチ・ステータスバーエリアを考慮
- **セーフエリア**: コンテナに `padding-top: env(safe-area-inset-top)` を設定し、ステータスバーとコンテンツが重ならないように
- **PWA起動時のスクロール**: 起動時に `window.scrollTo(0, 0)` を実行し、タブバーが画面上部に隠れないよう調整
- **ハンバーガーメニュー**: タブバー内に絶対配置。タップイベントの競合を防ぐため `stopPropagation()` を実装。z-index を 9998/9999 に設定し前面に表示
- **ステータスバー**: `apple-mobile-web-app-status-bar-style: black-translucent` を使用
